# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2025-a1-27

### Added
- Initial release of Google Cloud Backup and DR Terraform module
- Support for backup vault creation with access restrictions
- Backup plan configuration with flexible scheduling
- Backup plan association for GCP resources
- Configurable backup windows and retention policies
- Support for HOURLY, DAILY, and WEEKLY backup schedules
- Minimum enforced retention duration enforcement
- Comprehensive outputs for vault, plan, and association resources
